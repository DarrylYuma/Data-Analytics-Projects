# Bikes Sales Analysis
This project is an analysis and dashboard visualization built using a mock dataset of sales in the US by the brand Adidas. The dataset is first cleaned and then a dashboard is created that produced the following visualizations:
- Average Income per Purchase: A bar chart which shows the average income of customers that purchased a bike in comparison to the ones that did not purchase a bike. The data shown is divided by gender. This visualization shows that customers with a higher income are more likely to purchases bikes, therefore a discussion in regards to bike prices can be considered.
- Purchased Bikes per Age Bracket: A line chart which shows the amount of bikes purchased based on customer age brackets. This visualization shows that customers in Middle Age bracket are more likely to purchase bikes compared to the other age brackets, therefore it is advisable to cater more to this age bracket.
- Purchased Bikes per Customer Commute: A line chart which shows the amount of bikes purchased based on customer commute. This visualization provides insight that bikes are usually purchased by customers that have shorter commutes

The created dashboard can be interacted with to provide further insight based on marital status, regions, and education with the use of slicers. In summary, the dashboard created in this project allows the user to analyze and make appropriate business decisions using this dashboard.
